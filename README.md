# my-profile

このサイトは GitHub Pages で公開できます。

## 公開手順
1. GitHub のリポジトリで Settings > Pages を開きます。
2. Source を "Deploy from a branch" に設定します。
3. Branch で main を選択し、/ (root) を指定します。
4. 保存すると、次の URL で公開されます。
   - https://arisaharada770-rgb.github.io/my-profile/

## 公開対象
- index.html をトップページとして表示します。
- 各ページは相対リンクで接続されています。
